INTRODUÇÃO:
-----------
Projeto desenvolvido em parceria com a Altus sob projeto Finep...


INSTALAÇÃO:
-----------

Adicione os seguintes diretórios ao path do windows:

[INSTALL_DIR]\ExternalLibraries\OpenCV2.4.9\build\x86\vc11\bin
[INSTALL_DIR]\ExternalLibraries\AVTVimba_1.3\VimbaCPP\Bin\Win32
[INSTALL_DIR]\ExternalLibraries\libmodbus-master\src\win32

onde [INSTALL_DIR] é o diretório do projeto.

O projeto utiliza o QtCreator como IDE (abrir projeto [INSTALL_DIR]\CvWorks.pro)
e foi testado com compilador MSVC2012 no Windows 8 e 10.
