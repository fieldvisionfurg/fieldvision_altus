Detailed Description

template<class TImg, class TObj>
class VisionCore::Abstractions::TrackerLogger< TImg, TObj >

Provides log functionalities for a Tracker. Stores results, processing times and statistics.

This class wraps a tracker and transparently provides log functionalities. When the updateTracker function is called, it calls the updateTracker method of its internal tracker and:

  • Store the results.
  • Compute the processing time for tracking.
  • Compute statistics
  • Save the results to file
