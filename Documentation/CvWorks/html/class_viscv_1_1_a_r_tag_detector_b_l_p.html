Detailed Description

Implements a augmented reality tag detector.

This algorith is similar to the one implemented in ARToolkit. It first detects a bounding polygon around a tag. Then it extract the image inside the tag and match it agains a set of enrolled tags (see enrollTag method), assigning the tag with higher similarity.
